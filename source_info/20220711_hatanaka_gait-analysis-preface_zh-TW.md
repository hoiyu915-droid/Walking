## 作者對本書定位的主張

1. 本書想補足「涵蓋步行全貌」的臨床導向文本。作者原先想到 Perry 的 *Gait Analysis* 與 *Human Walking*，但認為僅靠個人能力難以超越這兩部經典；在出版社建議下，改以專業理學治療師為主要讀者。
2. 內容刻意納入較多臨床工作所需資訊，目標是在專業深度、可讀性與實用性之間取得平衡。
3. 對初學者而言，不建議像一般教科書般從第一頁順讀到最後一頁；應先利用導讀掌握領域全貌、區分基礎與應用，並依建議順序進入。
4. 對臨床專家，作者嘗試收錄其他教材較少涵蓋、但有臨床價值的內容。

## 可轉用到本 repo 的原則

### 1. 量測不是終點；機制理解才是

作者從步行分析研習的經驗提到：早期即使辛苦完成量測，仍可能被指出「並沒有正確量到」。真正關鍵不在得到一串量測數值，而在理解這些數據指向的人體運動機制；量測與分析是通往臨床應用的工具。

- 不把時空參數、關節角度、地面反作用力或其他 proxy 當成結論。
- 每個指標都應連回：量測品質 → 動作模式 → 可能機制 → 臨床可驗證的假設。
- 資料品質不足時，應明確標記「不能據此推論」，而不是用看似精密的圖表填空。

### 2. 步行分析的核心產出是臨床推論

作者將「理解機制，才能連到臨床應用」視為研習意義，並將其連結到 clinical reasoning（臨床推論）。

- 輸出應優先回答：此動作現象可能由什麼造成、有哪些競爭解釋、下一步如何區辨。
- 避免把分析退化為「數值異常清單」或單一路徑的病因宣告。
- 將生物力學定位為臨床決策工具，而不只是研究描述語言。

### 3. 動作觀察不可被檢查堆疊取代

作者認為臨床上可能正在增加對理學治療沒有幫助的檢查與評估；相對地，病人的動作不會說謊，而每位需要治療、存在動作異常的病人都與動作問題相關。

- 先建立真實任務中的動作描述，再用檢測或儀器資料驗證，而非反過來用靜態檢查取代步行本身。
- 必須保留任務、環境、速度、疲勞、疼痛與輔具等脈絡，否則「異常」可能只是測試情境產物。
- 這不等於儀器量測無用；原意是要把量測正確地接回機制與實作。

### 4. 臨床人員應看得比患者自己更細

作者期待理學治療師能理解患者動作的細小差異。分析框架需同時支援：

- 全局模式：步態週期、節律、對稱性、任務策略。
- 局部細節：關節／節段時序、代償、接觸與離地、軀幹與上肢參與。
- 可溝通性：將複雜發現轉成患者能執行、能追蹤的處置或下一步評估。
