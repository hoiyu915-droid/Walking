# 04｜筋膜動力學、ECM 與「微型黏連」的乾淨說法

## 結論

「微型黏連」這個詞很直覺，但容易被誤解成手術後 adhesion 的縮小版。較穩的語言是：

- 微尺度界面滑動受限
- ECM 緻密化
- myofascial interface restriction
- microfibrotic tethering
- fascial densification

核心不是「兩塊肉被黏死」，而是組織界面失去滑順剪切。

## 跟手術後 adhesion 的差異

### Surgical adhesion

通常指原本應該分開的組織表面之間，形成可辨識的纖維橋接或異常連結。

常見語境：

- 術後腹腔/骨盆沾黏
- scar band
- arthrofibrosis
- capsular fibrosis
- 明確創傷後纖維連結

這是宏觀結構問題。

### 這輪討論的「黏、硬、卡」

更像微尺度的界面問題：

- 筋膜層之間滑不順
- 肌肉內 ECM 緻密化
- 肌腱/筋膜/肌肉界面剪切順應性下降
- scar interface 不是大沾黏，但局部 tethering 增加
- hyaluronan / ground substance 黏滯性改變
- collagen packing / cross-linking 增加
- 被動 stiffness 上升

## 四級描述

### 1. 黏滯化：viscous glide restriction

偏向 hyaluronan / ground substance 黏滯性改變。

特徵：

- 動一動或熱開後變順
- 滑不開但不一定是真纖維橋
- 比較可逆

### 2. 緻密化：ECM / fascial densification

組織含水、hyaluronan、collagen 排列、局部壓縮狀態改變。

特徵：

- 層間滑動差
- 剪切順應性下降
- 角度進入某區間會卡

### 3. 微纖維 tethering：microfibrotic tethering

更接近 collagen deposition / cross-linking / fibroblast / myofibroblast 參與。

特徵：

- 小尺度纖維拉住
- 被動張力高
- 活動度需要更高機械刺激才可能改變

### 4. 真正黏連：scar adhesion / surgical adhesion

術後或明顯創傷後的結構性纖維連結。

特徵：

- 病理語義重
- 不能隨便套到一般「筋膜卡」
- 不應用手法話術直接替代影像或臨床診斷

## 最穩的段落寫法

這裡說的不是手術後那種宏觀沾黏，而是微尺度的界面滑動受限：筋膜、肌腱、肌肉 ECM 或 scar interface 因 hyaluronan 黏滯化、ECM 緻密化或低度纖維化，導致層間剪切滑動變差。臨床上感覺像黏、硬、卡，但不等於已證明有可見的 adhesion band。

## 與高負荷活動度的連接

高負荷可能讓原本滑動受限的 myofascial/tendon interface 進入足夠剪切與張力區，產生：

- 短期神經保護降低
- 層間滑動感改善
- 關節定位改變
- ECM turnover/remodeling window
- movement option 重開

但這不等於證明「黏連被撕開」。
