# 05｜運動營養：mito / Ca2+ / VitD / ECM 修復品質

## 結論

這條線比較適合用「運動醫學 + 運動營養」來講，而不是一開始就掉進臨床復健語境。

核心不是「某病怎麼治」，而是：

能量系統出問題 → Ca2+ handling 出問題 → 收縮/放鬆/疲勞/修復變差 → collagen / ECM / mineral turnover 變差 → 活動度、疼痛、步態、骨盆控制受影響。

## mito 與 Ca2+ handling

骨骼肌收縮靠 Ca2+ 訊號。

收縮後要放鬆，需要把 Ca2+ 泵回 SR，這需要 ATP。

mitochondria 一邊供應 ATP，一邊參與 Ca2+ buffering / uptake。

所以 mito 出問題不是只有「能量少」，也可能讓：

- Ca2+ release / reuptake 品質下降
- 收縮放鬆節奏變差
- 疲勞上升
- 肌張力變怪
- 恢復速度下降

## 運動營養的橋

運動營養不是只算熱量與蛋白質，而是在管理：

- ATP 供應
- 肝醣狀態
- Ca2+ handling
- VitD / VDR signaling
- Mg / Ca / P / PTH / 腎功能
- collagen / ECM 修復材料
- 發炎與恢復狀態

## VitD 不要硬接鈣化，應先接 ECM 與修復

VitD 接「肌肉鈣化」最硬的是過量/活性過高 → Ca/P 失控 → 軟組織鈣化。

但一般運動營養語境更適合接：

VitD / VDR 狀態差 → muscle Ca2+ handling、myogenesis、mitochondrial regulation 變差 → 修復品質下降 → TGF-beta / fibroblast / collagen 軸偏 fibrogenesis → ECM 堆積、滑動差、硬卡感。

## VitD / VDR 與 ECM 線

較穩模型：

VitD 不足或 VDR signaling 差
→ muscle Ca2+ handling、cell cycle、myogenic regulatory factors 下降
→ 肌肉修復與再生品質變差
→ 反覆微損傷後更容易走向 fibroblast / myofibroblast / ECM remodeling
→ collagen I、alpha-SMA、fibrotic tone 上升
→ 筋膜/肌腱界面/scar interface 滑動差
→ 黏、硬、卡、活動度受限。

## mito 與骨質 turnover

骨 remodeling 不是只有荷爾蒙問題，還是細胞能量問題。

mitochondria 可影響：

- osteoblast 的 matrix production / collagen synthesis / mineralization
- osteoclast 的分化、融合與骨吸收
- Ca2+ / phosphate microdomain
- mitophagy / mitochondrial dynamics
- bone turnover balance

較穩說法：mitochondrial dysfunction 可直接改變骨細胞功能，進而影響 bone turnover。

## mito 與鈣化

證據強度分層：

### 較強

- 骨 turnover
- 血管鈣化，尤其 CKD / vascular calcification 線

### 中等

- 正常礦化與 Ca-P handling
- matrix vesicles / mineralization / Ca2+ microdomain

### 待補證

- 肌腱鈣化 / calcific tendinopathy
- 肌肉內異位鈣化與 mito 的直接因果

不要說：mito 壞掉就一定鈣化。

比較穩的是：mitochondrial dysfunction 可能改變礦化與異位鈣化傾向，尤其透過 ATP、Ca2+、phosphate handling、mitophagy 與細胞命運改變。

## 與步態/骨盆控制的連接

代謝/營養狀態不是只影響能量，而是會改變組織可載入性與可恢復性。

可能鏈條：

mito / ATP / Ca2+ / VitD / ECM 修復品質下降
→ 筋膜、肌腱、關節囊、scar interface 材料性質變差
→ 層間滑動差、被動張力高、活動度受限
→ 骨盆控制與臀肌上線變差
→ 步態代償與疼痛威脅上升
→ 活動量下降、機械訊號品質下降
→ 恢復更差。

## 一句話

你不是在講復健小技巧，而是在講「代謝環境如何改變身體可承重性」。
